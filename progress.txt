### Design Decisions
- SQLite for persistence (single mini PC, no need for separate DB container)
- JWT for temporary links (self-contained, blacklist in memory only)
- No transcoding anywhere in the pipeline (H.264 cameras -> remux to HLS)
- MQTT for H/W communication, HTTP for inter-service S/W communication
- Mobile-first web UI with bottom tab navigation
